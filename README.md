# CarMod - Araç Bakım, Modifiye ve Takip Sistemi 🚗

Modern ve kullanıcı dostu araba bakım, modifikasyon ve parça takip uygulaması.

## 🖥️ Canlı Demo

`index.html` dosyasını tarayıcınızda açarak uygulamayı kullanabilirsiniz.

## ✨ Özellikler

- 🚗 **Garaj Yönetimi** - Birden fazla araç kaydı ve yönetimi
- 🔧 **Akıllı Bakım Rehberi** - Kilometre bazlı otomatik bakım takibi
- ⚡ **Modifikasyon Günlüğü** - Yapılan tüm modifikasyonların görsel kaydı
- 🛒 **Parça Marketi** - Araca uyumlu yedek parça kataloğu ve sepet
- 💻 **Yazılım & Dyno** - ECU/TCU yazılım ve beygir güç takibi
- ⛽ **Yakıt & Seyahat Takibi** - Yakıt tüketimi ve seyahat logları
- 📄 **Resmi Belgeler** - Muayene, kasko, sigorta son tarih takibi
- 📊 **Grafikler** - Harcama dağılımı ve yakıt eğilim grafikleri

## 🛠️ Teknolojiler

- **HTML5** - Semantik yapı
- **CSS3** - Glassmorphism tasarım, animasyonlar
- **JavaScript (Vanilla)** - Sıfır bağımlılık
- **Chart.js** - Grafikler
- **LocalStorage** - Veritabanı (sunucu gerekmez)

## 🚀 Kurulum

1. Bu repoyu klonlayın:
```bash
git clone https://github.com/KULLANICI_ADINIZ/carmod.git
```

2. Klasöre gidin:
```bash
cd carmod
```

3. `index.html` dosyasını tarayıcınızda açın. Hepsi bu kadar!

> 💡 Sunucu gerekmez. Tüm veriler tarayıcınızın LocalStorage'ında saklanır.
